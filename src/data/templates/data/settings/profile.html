If you would like to stop using the platform, you have a number of options:

  • You can simply not use the platform any more and uninstall any client applications, like the Brocrobes Chrome Extension, this website, etc. In this case your existing data will continue to be used in the active university research programme. You can always start using the platform again if you change your mind.
  • You can ask for your account to be completely deactivated by contacting Anton MELSER, anton.melser@my.cityu.edu.hk. In this case your existing data will continue to be used in the active university research programme. You can always start using the platform again if you change your mind by writing to Anton MELSER and requesting your account be reactivated.
  • You can ask for your account to be completely deleted by contacting Anton MELSER, anton.melser@my.cityu.edu.hk, including any existing data generated. This is the nuclear option, and all data will be irreversibly deleted. Except in very exceptional circumstances this will mean the end of your participation in the research, though you are obviously more than welcome to download and run the system yourself on your own resources!
